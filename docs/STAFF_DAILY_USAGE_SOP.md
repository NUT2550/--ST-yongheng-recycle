# Staff Daily Usage SOP — ระเบียบปฏิบัติประจำวัน

> สำหรับพนักงาน (staff) ที่ใช้ระบบยงเฮง มหาชัย รีไซเคิล

---

## 1. การรับซื้อ (Buy)

### ขั้นตอน:
1. ล็อกอิน → กดแท็บ "รับซื้อ"
2. เลือกสินค้า (ค้นหาในช่องค้นหา)
3. ใส่น้ำหนัก (กก.) — ต้องมากกว่า 0
4. ใส่ราคา/กก.
5. กด "เพิ่มรายการ"
6. ตรวจทานยอดรวมในตะกร้า
7. ถ้าเครดิต → ติ๊ก "ซื้อเชื่อ"
8. กด "บันทึกใบรับซื้อ"
9. **ยืนยันใน dialog** — ตรวจยอดรวม + กด "ยืนยันสร้างบิล"
10. จดเลขบิล (BUY-2569-xxxxx)

### ข้อห้าม:
- ห้ามใส่น้ำหนักติดลบ
- ห้ามกดบันทึก 2 ครั้ง — รอ dialog ยืนยัน
- ห้ามแชร์รหัสผ่าน

---

## 2. การนำเข้า Excel (Import)

### ขั้นตอน:
1. กดปุ่ม "นำเข้า Excel"
2. เลือกไฟล์ .xls หรือ .xlsx
3. ระบบอ่านไฟล์ + แม่นตรงสินค้าอัตโนมัติ
4. ตรวจตาราง preview:
   - สินค้าที่แม่นตรง = เครื่องหมาย ✓ สีเขียว
   - สินค้าที่ไม่ตรง = เครื่องหมาย ⚠️ สีเหลือง
5. คลิก dropdown "เลือกสินค้า" เพื่อแก้ไขการแมพ (เลื่อนด้วยเมาส์ได้)
6. กด "เพิ่ม X รายการในตะกร้า"
7. ตรวจตะกร้า + บันทึกบิลปกติ

### ข้อห้าม:
- ห้าม import ไฟล์ที่ไม่ใช่ .xls/.xlsx
- ห้ามแก้ไฟล์ Excel ต้นฉบับ

---

## 3. การตรวจบิล (History)

### ขั้นตอน:
1. กดแท็บ "ประวัติ"
2. เลือกประเภท: รับซื้อ / ขาย / คัดแยก
3. ใช้ตัวกรองวันที่ (ถ้าต้องการ)
4. ขยายบิลเพื่อดูรายละเอียด
5. ดูเลขบิล + สถานะ:
   - บิลปกติ = ไม่มี badge แดง
   - บิลยกเลิก = badge แดง "ยกเลิกแล้ว" + จาง + ขีดทับ

---

## 4. การยกเลิกบิล (Cancel)

### เมื่อไหร่ต้องยกเลิก:
- กรอกสินค้าผิด
- กรอกน้ำหนักผิดมาก
- บิลซ้ำ

### ขั้นตอน:
1. ไปหน้าประวัติ
2. ขยายบิลที่ต้องการยกเลิก
3. กดปุ่ม "ลบ" (ถังขยะ)
4. ใส่เหตุผลการยกเลิก
5. กดยืนยัน

### สิ่งที่เกิดขึ้น:
- บิลถูกทำเครื่องหมาย "ยกเลิกแล้ว"
- สต็อกคืนอัตโนมัติ
- บิลยังอยู่ในประวัติ (ไม่หาย)
- ยอดรวมในแดชบอร์ดไม่นับบิลยกเลิก

---

## 5. ข้อห้ามของพนักงาน

- ห้ามแชร์รหัสผ่าน
- ห้ามลบบิลถาวร — ใช้ "ยกเลิก" เท่านั้น
- ห้ามแก้บิลเก่าเกิน 7 วัน โดยไม่แจ้ง admin
- ห้ามจัดการผู้ใช้ / สินค้า / พนักงาน (admin เท่านั้น)
- ถ้ากรอกผิด → ยกเลิกบิล + สร้างใหม่
- ถ้าไม่แน่ใจ → ถาม admin ก่อนทำ

---

## 6. ถ้าเน็ตหลุด

- ข้อมูลที่กำลังกรอกอาจหาย
- รอเน็ตกลับ → refresh หน้า → ตรวจบิล
- ถ้าบิลไม่ขึ้น → สร้างใหม่
- ถ้าบิลขึ้นซ้อน → ยกเลิกบิลซ้อน
